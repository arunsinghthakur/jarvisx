# JarvisX Core

Shared Python library for JarvisX platform.

## Installation

```bash
pip install -e .
```

## Modules

- `jarvisx.a2a` - A2A utilities (discovery, storage, access)
- `jarvisx.common` - Common utilities (cache, logging)
- `jarvisx.config` - Configuration (configs, constants)
- `jarvisx.database` - Database models and session
- `jarvisx.mcp` - MCP server configs and loaders
- `jarvisx.services` - Shared services (email, knowledge base)
- `jarvisx.tracing` - LangFuse distributed tracing
- `jarvisx.workflows` - Workflow engine and nodes
