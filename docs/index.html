Getting Started

Up and Running in Minutes

Get started with JarvisX in four simple steps

1

Clone & Configure

2

Start Services

Run the start script to launch all services including agents, APIs, and the admin UI.

3

Create Workspace

Set up your organization and workspace with custom agent and MCP configurations.

4

Build & Deploy

Design workflows, chat with agents, and automate your tasks through voice or text.
